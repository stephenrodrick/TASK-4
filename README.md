Project Overview

This project demonstrates how data analysis and basic statistical methods can be used to validate business decisions and drive measurable impact. The objective was to evaluate whether a proposed business change (new website layout) led to a statistically significant improvement in conversion rate, and to present the findings in a clear, stakeholder-ready format.

The project follows a business storytelling approach, transforming raw analysis into actionable insights supported by statistical evidence.
